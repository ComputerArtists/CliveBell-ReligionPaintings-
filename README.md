Religiöse Generative Kunst – Sammlung 2025
Eine meditative Processing-Skript-Sammlung von uns beiden
Alle Skripte laufen in **Processing 4.x (Java-Modus) sofort ohne zusätzliche Bibliotheken.
Einfach Ordner anlegen → Skripte hineinkopieren → starten und staunen.
Inhalt der Sammlung (9 Skripte)

Mandala-Generator
Klassisches 12-fach symmetrisches Meditationsmandala, langsam rotierend und atmend.
Islamisches Sternmuster
Lebendiges, zufällig überlappendes Stern-Ornament (8-, 10-, 12-, 16-zackig) mit sanftem Atmen.
Taste N = neue zufällige Variante | S = speichern
Kreuz-Partikel-System
4200 leuchtende Partikel formen ein lateinisches Kreuz.
N = liturgische Farben wechseln (Gold, Blau, Violett, Rot, Grün, Weißgold) | S = speichern
Dharmachakra – Rad des Dharma
Buddhistisches 8-Speichen-Rad in 5 Stil-Varianten (Gold, Tibetisch, Zen, Lotus, Kosmisch).
N = Variante wechseln | S = speichern
Gebetsworte als Typografie-Wolke
Liest Worter.txt (eine Zeile = ein Wort) und lässt die Worte in Herz/Kreis/Spirale fliegen.
N = Form wechseln | R = Wörter neu laden | S = speichern
Bodhi-Blatt
Großes pulsierendes Bodhi-Blatt, aus dem ständig neue Blätter herauswachsen und davonfliegen.
N = Farbstimmung (Gold, Jade, Mondblau, Sonnenaufgang) | S = speichern
Endlos-Mantra
Liest Mantra.txt (eine Silbe pro Zeile) und lässt das Mantra endlos kreisen und atmen.
N = neues Mantra + neue Farben | S = speichern
Mandala der 1000 Buddhas
1008 kleine Buddha-Silhouetten erscheinen langsam in goldener Phyllotaxis-Spirale.
N = Farbvariante (Gold, Jade, Amethyst, Regenbogen) | S = speichern

Bedienung (gleich bei allen Skripten)

N oder n → neue Variante / neue Farben / neue Form
S oder s → hochauflösendes PNG mit Datum und Versionsnummer speichern
R oder r → (nur bei den Text-Skripten) Datei neu einlesen
Maus → oft zusätzliche Steuerung (Helligkeit, Tempo, Rotation)

Dateien, die du selbst anlegen kannst
Im jeweiligen Sketch-Ordner einfach eine Textdatei mit genau diesem Namen anlegen:

Worter.txt → für die Typografie-Wolke (ein Wort pro Zeile)
Mantra.txt → für das Endlos-Mantra (eine Silbe pro Zeile)

Beispiele liegen in den jeweiligen Ordnern als Vorlage.
Technische Hinweise

Alle Skripte nutzen P2D-Renderer und blendMode(ADD) für echtes Leuchten
Keine externen Bibliotheken nötig
Läuft auch perfekt als Fullscreen-Präsentation (strg + F in Processing)
Speicherort der Bilder: immer im gleichen Sketch-Ordner
